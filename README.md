## Install and run Rustlings

Rustlings is installed via Cargo:

### 1) Install Rustlings

```bash
cargo install rustlings
```

### 2) Initialize Rustlings

This creates a local `rustlings/` directory with the exercises:

```bash
rustlings init
```

### 3) Move into the Rustlings directory

```bash
cd rustlings
```

### 4) Start Rustlings

This will watch your progress and guide you through the exercises:

```bash
rustlings
```

Follow the instructions in the terminal to complete each exercise.
